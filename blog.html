What is DOM and what is the purpose of DOM?

Ans:Web documents can be programmed using the Document Object Model (DOM). By modeling the page, programs can alter the structure, appearance, and content of documents. DOM nodes and objects can be interacted with by programming languages.

What is event bubble?

Event The DOM has the idea of bubbling (Document Object Model). An event that is received by an element is bubbled up (or propagated) via the DOM tree through the element's parents and ancestors until it reaches the root element.


What is a callback function and why will you use it?

Callback functions are functions that are invoked within the outer function as parameters to complete a task or activity.
